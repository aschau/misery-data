# misery-data

Live data feed for the [Claude Developer Misery Index](https://www.raggedydoc.com/misery) — a real-time dashboard tracking Claude AI outage, model-quality, policy, and competitive-defection sentiment.

This repo holds the data files that the dashboard, Discord bot, and RSS subscribers fetch every 15 minutes:

- `current.json` — current misery score (0–10), source breakdown, social post counts, and recent incidents
- `feed.xml` — RSS feed combining all sources (Anthropic status + Reddit + Bluesky)
- `feed-official.xml` — RSS feed using the Anthropic status page only

The data is generated by the [Misery Index Discord bot](https://github.com/aschau/aschau.github.io/tree/master/tools/misery-index/discord-bot), which polls Anthropic's status page, r/ClaudeAI, r/ChatGPT, and Bluesky every 15 minutes, classifies each post, and writes the aggregated result here via the GitHub Contents API.

## Why a separate repo?

The dashboard's source code lives at [aschau/aschau.github.io](https://github.com/aschau/aschau.github.io). Splitting the data into its own repo lets the source-code repository go private without breaking the public-facing dashboard, RSS feeds, or anyone subscribing to the `raw.githubusercontent.com/aschau/misery-data/main/...` URLs.

## Endpoints

```
https://raw.githubusercontent.com/aschau/misery-data/main/current.json
https://raw.githubusercontent.com/aschau/misery-data/main/feed.xml
https://raw.githubusercontent.com/aschau/misery-data/main/feed-official.xml
```
